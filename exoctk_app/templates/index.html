ExoCTK is an open-source, modular, data analysis software pakcage focused primarily on atompspheric characterization of exoplanets and observation planning. This web application serves as your shortcut to using this software. Here we fold the functionality of the ExoCTK code into a lovely interface -- and it's yours for the using!

Among our weaponry are such diverse elements as :

  • Transit Light-Curve fitting tools (TLC)
  • Limb-Darkening Calculator (LDC)
  • IFS exoplanet spectra extraction (IFS)
  • Atomospheric Forward Models (AFM)
  • Bayesian Atmospheric Retrieval (BAR)
  • Planetary Atmospheres Libraries and tools (PAL)
  • Transit Observation Tools (TOT)
  • Time-series Observation and Reduction tools (TOR)
